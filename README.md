# 🏠 House Price Prediction with Uncertainty Quantification

This project is an interactive **Streamlit web application** that predicts house prices using **multiple linear regression** and provides **confidence intervals (CI)** and **prediction intervals (PI)** for uncertainty estimation.

---

## 🚀 Features

* 📊 Synthetic dataset generation
* 📈 Multiple Linear Regression (from scratch)
* 📉 Confidence Interval (CI) & Prediction Interval (PI)
* 🎯 Interactive prediction tool
* 📊 Model evaluation metrics:

  * R² Score
  * RMSE
  * MAE
* 📉 Residual analysis (Q-Q plot, histogram, etc.)
* 📚 Built-in statistical theory explanations

---

## 🧠 Key Concepts

* **Confidence Interval (CI):** Range for the *average prediction*
* **Prediction Interval (PI):** Range for *individual predictions*
* **Uncertainty Quantification** in machine learning models

## 📊 Model Overview

The model is based on **Ordinary Least Squares (OLS)**:

[
\hat{\beta} = (X^T X)^{-1} X^T y
]

It also computes:

* Residual Standard Error
* Confidence intervals
* Prediction intervals
